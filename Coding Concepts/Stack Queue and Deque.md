Operations :- 
- push() : element insertion, if stack is full overflow
- pop() : element deletion, if stack is empty underflow   
- isEmpty(): determines whether the stack is empty or not.
- isFull(): determines whether the stack is full or not.'
- peek(): returns the element at the given position.
- count(): returns the total number of elements available in a stack.
- change(): changes the element at the given position.
- display(): prints all the elements available in the stack. ^Ojnod5E3

Memory management: The memory is assigned in the contiguous memory blocks. The memory is known as stack memory as all the variables are assigned in a function call stack memory. The memory size assigned to the program is known to the compiler. When the function is created, all its variables are assigned in the stack memory. When the function completed its execution, all the variables assigned in the stack are released. ^TnxStBOr
