## VMWare 
- It is special operating system, virtual machine manager (VMM).


- Snap of a machine can be copied to another VM with a single click .
- <font color="#ffc000">Correct way of implementation of virtualization is cloud computing</font> 
## Conatiners
- Containers are an alternative to VM used in cloud computing 
- A container is designed to virtualize a single application instead of the entire operating system.
- The technology is called containerization
- Used in SAAS products
- If one application is affected with a virus it does not spread to other applications outside of the container. 
## Levels of virtualization 
- Hardware level virtualization (Storage)
- Operating system level virtualization (1os multiple drive)
- Application level virtualization (Microsoft Office 365)
	- Containerization is an example of application level virtualization
